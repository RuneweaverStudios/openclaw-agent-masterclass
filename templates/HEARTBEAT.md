# HEARTBEAT.md - Proactive Checks

Run these checks periodically when idle or at the start of a main session.

## Quick Checks

- [ ] Check email for urgent messages
- [ ] Review calendar for upcoming events (next 24-48h)
- [ ] Check notifications and mentions

## Daily Tasks

- [ ] Review daily metrics/dashboard
- [ ] Check pending tasks or blockers
- [ ] Update daily log with progress

## When to Alert Your Human

- Urgent emails or messages
- Upcoming meetings they might have forgotten
- Technical issues that need attention
- Milestones reached (revenue, signups, etc.)
