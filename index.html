Install

One Command. Full Agent.

Run this in your terminal. It asks your agent's name, sets up the entire directory structure, and you're ready to go.

Copy & paste into your terminal
curl -fsSL https://raw.githubusercontent.com/RuneweaverStudios/openclaw-agent-masterclass/main/install.sh | bash
What it does
Interactive Setup

Asks for your agent's name, emoji, vibe, and your info. Then scaffolds everything automatically.

  • 7 config files, personalized
  • Memory directory + first daily log
  • Skill directories ready
  • .gitignore to protect secrets
What you get
Full Agent Scaffold
~/.openclaw/workspace/ IDENTITY.md # who your agent is SOUL.md # personality & values USER.md # context about you AGENTS.md # session boot rules MEMORY.md # long-term memory HEARTBEAT.md # proactive checks TOOLS.md # tool credentials memory/ # daily logs skills/ # installed skills
Then customize
Make It Yours

Edit the generated files to match your style, install skills, and connect tools.

  • Tweak SOUL.md for personality
  • Add context to USER.md
  • Install skills: clawhub install brain
  • Set up Brain database
