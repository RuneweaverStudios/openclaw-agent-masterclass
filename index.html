INTRODUCTION

> BUILD AN AUTONOMOUS AI AGENT

This masterclass teaches you how to transform a fresh OpenClaw installation into an autonomous AI agent with persistent memory, specialized skills, and proactive automation capabilities. By the end, you'll have an agent that can remember everything, execute complex tasks, and operate proactively — just like ghost malone.

> WHAT YOU'LL LEARN

IDENTITY & PERSONALITY
15 MIN

  • USER.md — Human context
  • AGENTS.md — Session rules

MEMORY SYSTEMS
30 MIN
Build persistent memory that survives sessions

  • Essential skills (brain, smart-compact)

QUICK START

> GET STARTED NOW

1. CLONE THE REPO
git clone https://github.com/RuneweaverStudios/openclaw-agent-masterclass.git
cd openclaw-agent-masterclass
2. READ THE GUIDE
# Open the full markdown guide
cat GUIDE.md

# Or view in your browser
open MASTERCLASS.html
3. FOLLOW THE PHASES

Complete each phase in order. Each builds on the previous one.

  • Phase 1: Identity (15 min)
  • Phase 2: Memory (30 min)
  • Phase 3: Skills (45 min)
  • Phase 4: Automation (20 min)
  • Phase 5: Tools (30 min)
  • Phase 6: Testing (15 min)
4. VERIFY SETUP
# Test Brain (semantic memory)
cd ~/.openclaw/workspace/skills/brain
node scripts/capture-rest.mjs "Test: Setup complete"

# Test Smart Compact
node skills/smart-compact/scripts/compact.mjs
