# Daily Operations

Morning report on all VibeStack metrics.

## Runs

Every morning at 8am via cron

## Reports

- Waitlist count
- Revenue (MRR)
- Pending content
- Recent activity

Check ~/.openclaw/workspace/logs/daily-ops.log
